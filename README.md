# Logistic_Regression_model
# 🎯 Student Pass/Fail Prediction using Logistic Regression

This project uses **Logistic Regression** to predict whether a student will **pass or fail** based on their study hours and attendance.

---

## 📌 Project Overview

The goal of this project is to build a classification model that can analyze student behavior and predict academic performance.

---

## 📊 Dataset Features

- Hours_Studied → Number of hours a student studies  
- Attendance → Percentage of class attendance  

### 🎯 Target Variable
- Result (0 = Fail, 1 = Pass)

---

## 🧠 Model Used

- Logistic Regression (Binary Classification)

---

## ⚙️ Technologies Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

---

## 🔍 Steps Performed

1. Loaded and cleaned the dataset  
2. Split data into training and testing sets  
3. Trained Logistic Regression model  
4. Evaluated model using accuracy score  
5. Visualized data using bar chart and scatter plot  
6. Analyzed relationship between features and result  

---

## 📈 Results

- Train Accuracy: **1.0**  
- Test Accuracy: **1.0**

> Note: The model achieved perfect accuracy because the dataset is small and linearly separable.

---

## 📊 Visualizations

- Bar Chart (Pass vs Fail)  
- Pie Chart (Distribution of results)  
- Scatter Plot (Study Hours vs Result)  

---

## 🧾 Conclusion

Higher study hours and better attendance significantly increase the chances of passing. Logistic Regression performs well on this dataset due to its clear pattern, but real-world datasets are usually more complex.
